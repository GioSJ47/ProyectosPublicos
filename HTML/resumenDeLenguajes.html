Declaracion de Variables:

Las variables se usan para almacenar informacion (valores numericos, digitos, cadenas de texto y objetos) en memoria. Para especificar sus identificadores (nombres) se permite el uso de letras en minuscula, mayuscula, numeros a partir del segundo dígito (un identificador no puede comenzar con un número), los signos especiales $, _, ñ y las letras con acentos.

var identificador; //VARIABLE VACIA var a, b, c; //MULTIPLES VARIABLES VACIAS var identificador = 12; //VARIABLE CON VALOR ENTERO var a = 14, b = 'Hola', c = "Chao"; //MULTIPLES VARIABLES CON VALORES

Dimenciones de elementos:

Uno pensaría que con elemento.style.width; o con elemento.style.height; podemos averiguar las dimenciones de un elemento, pero no es así a menso que dichas dimenciones estén especificadas dentro del atributo style del elemento HTML, por ejemplo ;;div style="height: 10px; width: 20px;">, en ese caso si va a funcionar. Pero si las dimenciones de un elemento están especificadas desde un archivo CSS, se tendrá que hacer uso de elemento.clientWidth; y de elemento.clientHeight;.

CSS:

Invertir horizontalmente un elemento:

Se le puede aplicar a cualquier elemento, como puede ser ina imagen, texto y divs.

.elemento { -moz-transform: scaleX(-1); /* Firefox */ -o-transform: scaleX(-1); /* Opera */ -webkit-transform: scaleX(-1); /* Chrome y Safari */ transform: scaleX(-1); /* w3org */ filter: FlipH; /* Internet Explorer */ }

Invertir verticalmente un elemento:

Se le puede aplicar a cualquier elemento, como puede ser ina imagen, texto y divs.

.elemento { -moz-transform: scaleY(-1); /* Firefox */ -o-transform: scaleY(-1); /* Opera */ -webkit-transform: scaleY(-1); /* Chrome y Safari */ transform: scaleY(-1); /* w3org */ filter: FlipV; /* Internet Explorer */ }
